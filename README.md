## ✨ О проекте

Элегантный одностраничный сайт, посвящённый Международному женскому дню 8 Марта. Сайт включает:

- **🎬 Hero-секция** — красивая главная страница с анимированными элементами
- **📜 История праздника** — интерактивный таймлайн ключевых событий 1857–1917 годов
- **💝 Поздравление** — искренние слова с анимированным сердечком
- **📱 Адаптивный дизайн** — прекрасно выглядит на любых устройствах

## 📦 Установка и запуск локально

### Требования
- Node.js 20+

### Шаги

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/username/8-marta-greeting.git
cd 8-marta-greeting

# 2. Установите зависимости
npm install

# 3. Запустите сервер разработки
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

### Другие команды

```bash
# Сборка для продакшена
npm run build

# Предпросмотр собранной версии
npm run preview

# Проверка типов TypeScript
npm run lint
```

## 🎨 Кастомизация

### Замена изображений

Положите свои изображения в папку `public/images/` с такими именами:
- `hero.jpg` — для главной секции
- `message.jpg` — для поздравления
- `history1.jpg` — `history4.jpg` — для исторических событий

### Изменение текстов

Тексты находятся в компонентах:
- [Hero.tsx](src/components/Hero.tsx) — главный заголовок
- [History.tsx](src/components/History.tsx) — исторические события
- [Message.tsx](src/components/Message.tsx) — текст поздравления
- [Footer.tsx](src/components/Footer.tsx) — текст подвала

## 🌐 Деплой на GitHub Pages

Проект настроен для автоматического деплоя на GitHub Pages через GitHub Actions.

1. Создайте форк этого репозитория
2. В настройках репозитория: **Settings** → **Pages** → **Source: GitHub Actions**
3. Сделайте любые изменения и запуште в `main`
4. Сайт автоматически задеплоится!
